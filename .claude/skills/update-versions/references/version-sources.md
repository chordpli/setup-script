# Version Sources

Official sources for checking the latest versions of each supported AI service.

## Claude Desktop

- **Release info**: https://claude.ai/download
- **macOS**: DMG download from official site
- **Windows**: EXE installer from official site
- **Linux**: Debian package / snap
- **Version detection**: Web search for "Claude Desktop latest version" or check download page

## ChatGPT / Codex (OpenAI CLI)

- **GitHub**: https://github.com/openai/codex
- **Version API**: `https://api.github.com/repos/openai/codex/releases/latest`
- **npm**: `npm view @openai/codex version`
- **Version detection**: GitHub Releases API (preferred) or web search

## Gemini CLI

- **GitHub**: https://github.com/google-gemini/gemini-cli
- **Version API**: `https://api.github.com/repos/google-gemini/gemini-cli/releases/latest`
- **npm**: `npm view @anthropic-ai/gemini-cli version` (check actual package name)
- **Version detection**: GitHub Releases API (preferred) or web search

## Ollama

- **GitHub**: https://github.com/ollama/ollama
- **Version API**: `https://api.github.com/repos/ollama/ollama/releases/latest`
- **Official site**: https://ollama.com/download
- **Version detection**: GitHub Releases API (preferred) or web search
